## Artillery 부하테스트

```
npm install -g artillery
```

- yml 파일로 테스트 시나리오 작성 가능
- 테스트 결과 그래프 시각화 지원
- HTTP, websocket, gRPC 등 지원

JMeter 보다 간단하게 테스트 가능할 듯
